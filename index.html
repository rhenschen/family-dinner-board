Google Calendar Integration
Connect a Google Calendar to automatically block family members from voting when they have work shifts that overlap with 5:00 PM or later.
Google Calendar ID
Find this in Google Calendar → Settings → your calendar → "Integrate calendar" → Calendar ID. The calendar must be set to public with "See all event details" (not just "See free/busy") under "Access permissions for events".
Google API Key
Create one at Google Cloud Console with the Google Calendar API enabled. Leave blank if you set GOOGLE_CALENDAR_API_KEY as a Netlify env variable.
Keyword Matching
Shared calendar: Events are matched to family members using their work keyword or name (set in the Family tab). If an event title contains a member's name and the word "work", and the shift extends past 5 PM, that member is automatically marked as working.
Personal calendars: You can also set a Google Calendar ID per member in the Family tab. Any event on their personal calendar that extends past 5 PM will automatically block them — no keyword matching needed.
Calendar settings are shared with all devices. Sync updates the working status for all 14 days.
